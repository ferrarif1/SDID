SDID Demo dApp｜SDID 演示应用

Connect to the SDID browser extension, approve the login request, and verify the DID signature from this page.｜通过 SDID 浏览器扩展批准登录请求，并在页面中验证 DID 签名。

How it works｜工作流程

  1. Install the SDID extension and create at least one DID identity.｜安装 SDID 扩展并创建至少一个 DID 身份。
  2. Click “Connect with SDID” to send a login request to the extension.｜点击“连接 SDID”向扩展发送登录请求。
  3. Approve the MetaMask-style prompt and optionally remember this site.｜在类似钱包的确认弹窗中审批，并可选择记住此站点。
  4. The demo verifies the signature with the returned public key.｜演示站点使用返回的公钥验证签名。

Try it now｜立即体验

If you previously authorized this site, use “Force prompt” to see the approval dialog again.｜如果你已经授权过此站点，可点击“强制弹窗”重新查看审批流程。
